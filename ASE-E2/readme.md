项目描述：

    此包包括五个文件:
    menu.h --------------菜单接口头文件
    linkTable.h ---------链表接口头文件
    linkTable.c ---------链表操作函数具体实现
    menu.c --------------菜单操作具体视线
    menuTest.c ----------测试菜单接口调用

编译方法：
  
    gcc menu.c test.c linktable.c -o test
    .\test

使用指南：

    用户通过menu.h调用添加命令函数，将命令名称、命令描述、指向命令实现函数的指针作为参数传递进去，并同时
    定义对应命令的函数功能。本menu实现了命令的添加、删除功能。
